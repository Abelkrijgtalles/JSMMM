# JUST SHOW ME MY MAP!
Shows your map while rowing a boat. That's it. It only applies to maps, and not anything else (more configuration will come in the future).

This was made mostly for making it easier to use buried treasure maps. Now you don't have to stop to look if you are close to the buried treasure.
